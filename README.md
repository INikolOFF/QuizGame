A simple Python console-based quiz game to test your general knowledge.

What this project practices: using functions, loops, conditional statements, lists, tuples, string methods, and variables. It also practices user input handling and score tracking.

How to play: run the Python script, answer the questions by typing your response, and see your score at the end.

Features: gives immediate feedback (Correct ✔️ / Wrong ❌), keeps track of attempts, and calculates your total score.

Future Improvements: add more questions, add categories, include a timer, and save high scores to a file.
